## Install GoldenDict on Android

First things first, download the latest version of Digital Pāḷi Dictionary from [Github](https://github.com/digitalpalidictionary/digitalpalidictionary/releases){:target="\_blank"}.

Download and install GoldenDict from the [Google Play Store](https://play.google.com/store/apps/details?id=mobi.goldendict.android.free)

![google play store](https://user-images.githubusercontent.com/64521731/152176609-919f7067-4985-41e1-b309-febf206198bb.jpg)

Using any file manager, copy the DPD folder into the GoldenDict folder. You can find that in the root directory, or on your SD card if you have one.

![image](https://user-images.githubusercontent.com/64521731/152172120-961947a1-576b-4e23-9a62-fee5466bbd78.png)

Run the GoldenDict app. 

A message will come up telling you that the dictionary needs to be indexed. Click **Proceed**.

![image](https://user-images.githubusercontent.com/64521731/152172611-8dfc89e6-4bbc-450c-9d80-40b5a0399200.png)

When the indexing is finished, click **Done**.

![image](https://user-images.githubusercontent.com/64521731/152172715-85f00bb1-e6a0-4a93-abdc-534d65092358.png)

You're all set up!

You can access DPD at any time from any app by selcting a word, holding it until the options menu comes up, and clicking **Share > GoldenDict**.

![dpd in goldendict](https://user-images.githubusercontent.com/64521731/152176837-d75bcc3e-b584-46a9-b3b1-a8e9edd60226.jpg)

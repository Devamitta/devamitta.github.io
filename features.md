## Features

#### DPD recognises 1.8 million inflected forms of Pāḷi words
When you selct on a word in a Pāḷi text and press the hotkey, DPD automatically recognises all the headwords to which that inflected form belongs. For example, clicking on _takkarassa_ will automatically open _takkara_. 
![inflected forms](https://user-images.githubusercontent.com/64521731/151703120-0a658082-9c1b-4ef8-9b93-4148359d22ed.png)

#### Compact presentation
All the important Pāḷi information is presented in **one compact line**, available at a glance. 
![compact line](https://user-images.githubusercontent.com/64521731/151702298-793b7f96-b6a8-4fb3-b07b-72eabcf45c88.png)
More detailed information is available by clicking the buttons. Essential information includes the part of speech, case which the word takes, English meaning, literal meaning and simplified construction.

#### Grammar
Click the **grammar** button to get more more detailed grammatical information about the word.
![grammar](https://user-images.githubusercontent.com/64521731/151702375-2eb2e4ff-9f98-4946-82d2-137f41909f98.png)
This includes root information, detailed construction, derivative, phonetic changes, compound, antonyms, synonyms, non-Indo-Aryan cogantes, Sanskrit cogantes and Sanskrit root.
![grammar2](https://user-images.githubusercontent.com/64521731/151702589-383d25e9-b960-4e86-9621-6c0141450cf7.png)
If you spot a mistake, please click the link to correct it.

#### Examples
Click the **examples** button to see relevant sutta examples of the word. 
![examples](https://user-images.githubusercontent.com/64521731/151702639-a10fa19f-063b-4718-9cff-aafb1151a7a3.png)
If you can think of a better example, please let me know!

#### Declension / Conjugation
Click the **declension** or **conjugation** button to see all the inflected forms of a word. 
![declension](https://user-images.githubusercontent.com/64521731/151702803-acdce09d-8c78-48fd-a61e-7a348c3af190.png)
This table is automatically generated using a stem + pattern system, e.g. anicc + o. Irregular inflections have their own unique tables.  

#### Root Family
Click the **root family** button to see all words which have the same prefix + root.
![root family](https://user-images.githubusercontent.com/64521731/151703328-969e8840-7419-4240-a8db-f9ef9f1f8d36.png)
This information is unique to DPD and is based on a detailed study of the Pāḷi and Sanskrit root system. It's great for getting to know the nuanced range of meanings that a root + prefix combinations can display.

#### Compound family 
Click the compound family button to see a collection of all the compounds which contain a particular word.
![compound family](https://user-images.githubusercontent.com/64521731/151703490-64216f0b-333b-4fe5-9729-9737960e0b3e.png)
If the headword itself is a compound, all compound families of the component words will be listed.
![compound family2](https://user-images.githubusercontent.com/64521731/151703612-07fdebcf-138e-48fa-ae2e-5461b8beba0b.png)

#### Roots Dictionary
There is a detailed **Pāḷi Roots Dictionary** included in DPD. Clicking on any root will take you to a special page containing all the root information.
![roots dict1](https://user-images.githubusercontent.com/64521731/151703830-be0690b5-b5ad-4c2c-bebb-dee7813e2653.png)
There you will find comprehensive information from classical grammars, 
![roots dict2](https://user-images.githubusercontent.com/64521731/151703862-6d4090df-e760-4c97-8e0a-289f89d2b0eb.png)
and detailed lists of all the root families.
![roots dict3](https://user-images.githubusercontent.com/64521731/151703904-d1338275-0479-478c-a3ce-27aa41cfe4db.png)

#### Sinhala and Devanagari Scripts
Good news for those who read Pāḷi in Sinhala or Devanagara scripts, DPD recognises all inflected forms in both those scripts.
![devanagari](https://user-images.githubusercontent.com/64521731/151704001-2232cd80-2fba-4650-94d1-523ea31a125c.png)
Transliteration is based on [Aksharamukha Script Converter](https://aksharamukha.appspot.com/about).  

#### Feedback
One of the most important aspects of DPD is the feedback loop. When you spot a mistake, please click the link to correct it. That opens up a Google form which is prefilled with the headword. It generally takes less than a minute to log an error, and helps to improve the quality of the dictionary for everyone who uses it. 
![feedback](https://user-images.githubusercontent.com/64521731/151704651-03a75e3c-fa31-4b31-b1d1-f3d1e55c2bd9.png)

## Fonts
If you prefer a sans-serif font, please install [Noto Sans](https://fonts.google.com/noto/specimen/Noto+Sans), it works well with Pāḷi diacritics. If you prefer a serif font, please install [Deja Vu Serif](https://www.fontsquirrel.com/fonts/dejavu-serif). or [Verajja Serif](https://en.m.fontke.com/font/10186264/), a specially created font for Pāḷi text.
